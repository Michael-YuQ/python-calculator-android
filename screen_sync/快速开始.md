# 快速开始指南

## 场景：从 Windows 发送屏幕到另一台 Windows

### 步骤 1: 准备服务器（可以是任意一台电脑或云服务器）

如果使用本地测试，在发送端电脑上运行：

```bash
python server.py
```

如果使用云服务器（CentOS）：

```bash
# SSH 连接到服务器
ssh user@your-server-ip

# 上传文件
scp server.py user@your-server-ip:~/

# 运行服务器
python3 server.py

# 或使用启动脚本
chmod +x start_server.sh
./start_server.sh
```

### 步骤 2: 启动发送端

在要分享屏幕的电脑上：

```bash
# 方法1: 使用脚本（推荐）
start_sender.bat

# 方法2: 手动运行
python client_sender.py localhost 9999 4

# 如果服务器在远程
python client_sender.py 192.168.1.100 5003 4
```

### 步骤 3: 启动接收端

在要查看屏幕的电脑上：

```bash
# 方法1: 使用脚本（推荐）
start_receiver.bat

# 方法2: 手动运行
python client_receiver.py localhost 5003

# 如果服务器在远程
python client_receiver.py 192.168.1.100 5003
```

## 场景：使用云服务器中转

### 1. 在 CentOS 服务器上部署

```bash
# 安装 Python（如果没有）
sudo yum install -y python3

# 上传 server.py
scp server.py user@your-server-ip:~/screen_sync/

# 启动服务器
cd ~/screen_sync
python3 server.py

# 后台运行
nohup python3 server.py > server.log 2>&1 &
```

### 2. 配置防火墙

```bash
# 开放端口
sudo firewall-cmd --permanent --add-port=5003/tcp
sudo firewall-cmd --reload

# 检查端口
sudo firewall-cmd --list-ports
```

### 3. 发送端连接

```bash
python client_sender.py your-server-ip 5003 4
```

### 4. 接收端连接

```bash
python client_receiver.py your-server-ip 5003
```

## 常见问题

### Q: 如何修改捕获区域？

A: 编辑 `client_sender.py`，修改 `get_screen_region()` 方法。

### Q: 如何提高画质？

A: 编辑 `client_sender.py`，修改 `quality=75` 为更高的值（如 90）。

### Q: 如何降低延迟？

A: 提高帧率参数，例如使用 `10` 或 `15` FPS。

### Q: 连接不上服务器？

A: 检查：
1. 服务器是否启动
2. 防火墙是否开放端口
3. IP 地址是否正确
4. 网络是否连通（ping 测试）

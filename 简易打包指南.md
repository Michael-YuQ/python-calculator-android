# Windows 打包 Android APK 简易指南

## 方案一：使用 GitHub Actions（最简单，推荐）

### 步骤：
1. 在 GitHub 上创建一个新仓库
2. 将项目文件上传到仓库
3. GitHub 会自动构建 APK
4. 在 Actions 标签页下载构建好的 APK

### 优点：
- 无需本地配置环境
- 完全免费
- 自动化构建

---

## 方案二：使用 Docker Desktop（需要安装 Docker）

### 前置要求：
1. 安装 Docker Desktop for Windows
   下载地址：https://www.docker.com/products/docker-desktop

### 构建命令：
```bash
docker run --rm -v "%cd%":/app kivy/buildozer android debug
```

---

## 方案三：使用 BeeWare（纯 Python 方案）

### 安装：
```bash
pip install briefcase
```

### 初始化项目：
```bash
briefcase new
```

### 构建 APK：
```bash
briefcase create android
briefcase build android
briefcase package android
```

---

## 方案四：完整配置 Windows 环境（复杂）

### 需要安装：

1. **Java JDK 17**
   - 下载：https://www.oracle.com/java/technologies/downloads/#java17
   - 安装后设置环境变量 JAVA_HOME

2. **Android Studio**
   - 下载：https://developer.android.com/studio
   - 安装 Android SDK
   - 设置环境变量 ANDROID_HOME

3. **Android SDK 组件**（在 Android Studio 中安装）：
   - Android SDK Platform 31
   - Android SDK Build-Tools
   - Android SDK Command-line Tools
   - Android NDK

4. **Python 依赖**：
```bash
pip install buildozer cython
```

5. **构建**：
```bash
buildozer android debug
```

---

## 推荐方案

对于 Windows 用户，我强烈推荐：
1. **GitHub Actions**（最简单）
2. **Docker Desktop**（次选）
3. **BeeWare**（如果想要跨平台）

完整环境配置比较复杂，不推荐新手使用。
